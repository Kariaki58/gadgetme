# **App Name**: StoreStack

## Core Features:

- Frontend Store Management: User sign-up and login, creating/loading distinct store data securely from LocalStorage, simulating multi-tenancy without a backend.
- Product CRUD & Details: Store owners can add, edit, or delete gadget products, setting details like name, cost/selling price, stock, category, and images. Each product generates a public page.
- Online Checkout Simulation: A frontend simulation for customer orders, including quantity selection. Reduces product stock, saves order data locally, and updates analytics.
- In-Person Sale Tracking: Tool for store owners to record in-person sales, automatically calculating expected amounts, extra charges, profit, and loss based on actual amount collected.
- AI Product Description Assistant: Leverage a generative AI tool to assist store owners in crafting compelling and unique product descriptions for their gadgets.
- SaaS Dashboard Analytics: Display key business metrics such as total revenue, profit, loss, overcharge, and orders using interactive charts (e.g., simulated monthly revenue, revenue breakdown).
- Public Store Catalog: A shareable public page displaying the store's products, with filtering by category and search functionality.

## Style Guidelines:

- Primary Color: #6B22CC (Deep Violet). A sophisticated, vibrant deep purple, suitable for primary brand elements and key interactions in a clean, modern interface.
- Background Color: #F7F1FF (Pale Lavender Mist). A very light, desaturated variant of the primary hue, providing an airy and pristine canvas for the dashboard, enhancing readability.
- Accent Color: #8989CE (Soft Periwinkle Blue). An analogous blue hue, brighter and less saturated than the primary, offering a calm yet distinct contrast for secondary actions, highlights, or informational elements.
- Headline and Body Font: 'Inter' (sans-serif). A modern, objective, and highly legible sans-serif font, ideal for a clean SaaS dashboard across all text elements.
- Utilize a consistent set of clean, line-based or filled icons that clearly represent common SaaS actions and data points, adhering to the overall minimalist design aesthetic.
- Employ a responsive SaaS dashboard layout with a clear sidebar navigation for desktop. This layout should adapt gracefully to single-column stacking for mobile and smaller screen sizes, ensuring full usability on all devices.
- Incorporate subtle and purposeful animations for user feedback (e.g., button clicks, form submissions) and smooth state transitions within the UI, avoiding excessive or distracting movement to maintain a 'simple, not overdesigned' feel.
